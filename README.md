
# 🛒 Online Shopping Cart (C++)

A simple and clean  C++ console application that simulates an online shopping cart.
The project demonstrates OOP , STL , formatted price output , coupon logic , and a clear separation of data and UI.

---

## 🚀 Features

- Add products to a shopping cart
- Realistic product prices in USD
- Quantity-based subtotal calculation
- Apply discount coupons (0–100%)
- Automatic price formatting (`$ xx.xx`)
- Edge-case handling (invalid coupons, zero quantity)
- Clean and readable code structure

---

## 📦 Project Structure

Files included:

- `main.cpp` — application logic and menu
- `datamodel.h` — classes: Product, Item, Cart
- `README.md` — project documentation

## 🛠️ Technologies Used

- **C++14 / C++17** (compatible with Code::Blocks 20.03)
- **STL** (`vector`, `string`, `iostream`)
- **OOP** (Product, Item, Cart)
- **Formatted numeric output** using `<iomanip>` and `<sstream>`

---

## 📥 How to Run

1. Open the project in **Code::Blocks 20.03** or any C++ IDE
2. Build and run `main.cpp`
3. Follow the menu to add products, apply coupons, and checkout
